供应商关系管理-激励和管理供应商绩效
发表于 2019-03-07| 更新于 2019-07-19|采购工作相关
阅读量:

促进供应商的承诺与合作

提高供应商的承诺、合作和忠诚度：

  1. 供应商激励与绩效管理，利用供应商激励和惩罚。
  2. 确保对绩效和关系的改进给予有意义的激励，且在达到时候给予奖励（包括非正式的认可、感谢和表扬）
  3. 与供应商维持积极的、加强关系的联络和沟通。（最好有专人负责对接）
  1. 获得双方高级经理的承诺和支持，必要时如冲突或争议等情况得到高层的支持。
  2. 培养个人联系和人脉，建立各方的信任和商誉。
  3. 尽可能进行“双赢”谈判，表现出理解供应商在问题和谈判中的观点。
  4. 通过保持足够的业务量，使供应商对合作进行投资，并且通过良好的合作，信誉等方面做一个富有吸引力的买家。（按时付款、公平处理争议、共担风险、公平获益等）
  5. 让供应商参与产品设计与共同开发、系统集成、收益共享等。
  6. 确保尽可能遵守互利互惠或双赢原则。

供应商奖励

供应商奖励措施举例，包括财务与非财务：

  1. 设定具体的与认可或奖赏挂钩的关键绩效指标：续签合同或承诺将来继续合作、纳入首选供应商名单、公开表扬、财务奖励。
  2. 分享收入、利润或收益。（供应商通过改进给买家创造更多价值、收入或利润，分一部分给供应商）
  3. 许诺长期合作或增加业务量。
  4. 订货量稳定或确定最低订货量，便于供应商安排投资和改进。
  5. 许诺长期合作和增加业务量，前提需要买家富有吸引力，比如按时还款。
  6. 创新机会：如可以让供应商有机会实施或创造新解决方案，利于供应商发展。
  7. 为产品开发提供支持。
  8. 实施供应商奖励计划，给绩效优秀供应商公开褒奖。
  9. 其他正面反馈，供应商绩效好，采购团队就要表示感谢和表扬。

激励原则

  1. 平衡，不能只强调某一方面导致其他方面受损害。例如通过牺牲质量来追求产量或降低成本则不可取。（尤其在制药业企业，质量权重远大于单纯成本，尤其在健康医药行业）
  2. 奖励标准要明确，不能过于狭隘，避免片面追求某方面成果扼杀灵活性或创新。
  3. 公平且易于监督。绩效标准明确，达到目标就要实施奖励。

供应商惩罚

  1. 绩效不佳威胁减小交易额。
  2. 排除出首选供应商名单外。
  3. 将糟糕供应商等级评分公之于众。（可以看做是发布于官方平台进行曝光）
  4. 在合同中明确，如因供应商原因未能按要求履行合同导致买方损失，买方可以提出索赔。

如果希望得到长期承诺、合作及提升，“胡萝卜”比“大棒”更管用。

供应商绩效测量与管理-（绩效评估作用）

正式的供应商绩效评估具有多方面意义。

  1. 帮助识别质量高、绩效优的供应商。（哪些供应商可获得特定订单、是否进入首选供应商名单、是否有潜力发展为更为密切的战略伙伴供应商、如何在若干供应商之间分配采购支出以降低风险）
  2. 有助于保证供应商履行合同承诺。
  3. 为供应商提供激励，以保持和持续改进其绩效水平。
  4. 通过识别问题及其原因或识别需要得到支持和开发，从而提升绩效。
  5. 供应商评估与供应商绩效评估不同。前者为合同签署前，评估是否具有满足买家需求的能力，后者评判现有供应商满足买家需求过程中的绩效。
